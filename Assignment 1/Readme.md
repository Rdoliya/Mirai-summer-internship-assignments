# 📡 Assignment 1 – Identity Echo Interface

## 📖 Overview

The Identity Echo Interface is a Streamlit application developed as part of the AI Builder Track internship.

It collects a user's name and message, validates the inputs, and displays a personalized response. The application also estimates the number of AI tokens that would be consumed based on the message length.

## ✨ Features

* User-friendly Streamlit interface
* Name and Message input fields
* Input validation
* Success, Warning, and Error messages
* AI Token Cost Estimator
* Clean and responsive UI

## 🛠️ Technologies Used

* Python
* Streamlit

## 🚀 Installation

Install Streamlit:

```bash
pip install streamlit
```

Run the application:

```bash
streamlit run app.py
```
