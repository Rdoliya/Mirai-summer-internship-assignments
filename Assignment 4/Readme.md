# 🎨 AI Image Studio

AI Image Studio is a Streamlit application that generates AI-powered images using Pollinations AI.

## ✨ Features

- AI Image Generation
- Multiple Art Styles
- Width & Height Controls
- Magic Enhance Mode
- Surprise Me Generator
- Download Images as PNG

## 🛠️ Technologies

- Python
- Streamlit
- Pollinations AI
- Requests

## 🚀 Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
