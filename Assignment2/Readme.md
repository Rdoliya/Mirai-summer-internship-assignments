# 🌍 Assignment 2 – AI Multiverse Chatbot

## 📖 Overview

AI Multiverse Chatbot is an interactive Streamlit application powered by Google's Gemini AI.

Users can select from multiple AI personalities and have dynamic conversations where the AI responds according to the selected character.

## ✨ Features

* 10 Unique AI Personalities
* Interactive Chat Interface
* Powered by Google Gemini API
* Session Chat History
* Clear Chat Button
* Modern Streamlit UI
* Secure API Key Management using `.env`

## 🎭 Available Personalities

* Common Indian Man
* Crazy Salman Khan Fan
* Little Boy
* Motivational Coach
* Software Engineer
* College Professor
* Stand-up Comedian
* Entrepreneur
* Friendly Teacher
* AI Assistant

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Gemini API (`google-genai`)
* python-dotenv

## 📂 Project Structure

```text
│── app.py
│── requirements.txt
└── README.md
```

## 🚀 Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
